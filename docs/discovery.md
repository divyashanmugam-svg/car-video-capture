# Locked MVP Decisions

## Product

- Standalone mobile web page
- ACKO motor claims use case
- Pre-capture education screens included
- Post-capture preview with delete and retake

## Capture

- Prefer rear camera
- Landscape-first experience
- Audio enabled
- Flash/torch toggle attempted where supported
- Soft nudge after recommended full walkaround is complete
- Minimum duration target: 90 seconds for MVP

## Guidance

- Timed prompts rather than CV-driven path detection
- Haptic feedback where available
- Warning states for permissions, orientation, and short capture
- Coachmarks cover a full exterior walkaround

## Metadata

- Timestamp
- Geolocation if allowed
- Device/browser metadata
- Basic session integrity/risk signals
